### [818. SET](https://projecteuler.net/problem=818)

The SET® card game is played with a pack of $81$ distinct cards. Each card has four features (Shape, Color, Number, Shading). Each feature has three different variants (e.g. Color can be red, purple, green).

A *SET* consists of three different cards such that each feature is either the same on each card or different on each card.

For a collection $C_n$ of $n$ cards, let $S(C_n)$ denote the number of *SET*s in $C_n$. Then define $F(n) = \sum\limits_{C_n} S(C_n)^4$ where $C_n$ ranges through all collections of $n$ cards (among the $81$ cards).
You are given $F(3) = 1080$ and $F(6) = 159690960$.

Find $F(12)$.

### 818. 神奇形色牌

一副神奇形色牌由 $81$ 张互不相同的纸牌构成，每张牌上的图案都有形状、颜色、数量、纹理四个特征，每个特征都有三种可能情况。例如，一张牌上图案的颜色可能是红色、紫色或绿色。

若这副牌中不同的三张牌满足：这三张牌的每项特征要么完全相同，要么互不相同，则称这三张牌构成一个 *牌组*。

对一个由 $n$ 张牌组成的牌集 $C_n$，记其中所含的 *牌组* 数量为 $S(C_n)$。然后记 $F(n) = \sum\limits_{C_n} S(C_n)^4$，其中 $C_n$ 取遍这副牌中任何 $n$ 张牌的组合。已知 $F(3) = 1080$、$F(6) = 159690960$。

求 $F(12)$。
