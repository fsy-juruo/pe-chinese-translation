955. Finding Triangles

A sequence (an)n≥0 starts with a0=3 and for each n≥0,

  • if an is a triangle number (A triangle number is a number of the form m(m+1)/2 for some integer m), then an+1=an+1;

  • otherwise, an+1=2an−an−1+1.

The sequence begins:

3,4,6,7,9,12,16,21,22,24,27,31,36,37,39,42,…

where triangle numbers are marked red.

The 10th triangle number in the sequence is a2964=1439056.
Find the index n such that an is the 70th triangle number in the sequence.

955. 寻找三角形数

我们按如下规则构造序列 (an)n≥0：a0=3，且对诸 n≥0：

  • 若 an 是一个 三角形数（可被写作 m(m+1)/2,m∈Z 的数），则 an+1=an+1。

  • 否则，an+1=2an−an−1+1。

该序列的前 16 项如下，其中三角形数已被标红：

3,4,6,7,9,12,16,21,22,24,27,31,36,37,39,42,…

这个序列中的第 10 个三角形数是 a2964=1439056。
求这个序列第 70 个三角形数 an 的下标 n。
